## Description

üîç A Library app made using node.js, express.js, mySQL, Sequelize and handlebars.js that allows users to track their inventory of collected books and socially interact with other book lovers
  


## User Story

```
As a collector of quality books, I want to track my collection, add personal observation, share my collection with others, and search for quality books to add to my collection.
```
